Connecte ton HubSpot
Entre ton token d'accès HubSpot pour permettre à l'extension d'importer les prospects.
📋 Comment obtenir ton token :
  1. Connecte-toi sur app.hubspot.com
  2. Clique ton avatar (haut droite) → Intégrations
  3. Dans le menu gauche → Applications privées
  4. Clique Créer une application privée
  5. Onglet Portées → coche crm.objects.contacts.read et crm.objects.contacts.write
  6. Clique Créer → copie le token affiché
Le token commence par pat-eu1- (Europe) ou pat-na1- (USA)

Comment gérer les contacts déjà présents dans HubSpot ?
✓
Extension prête !
Prospector est connecté à ton HubSpot. Tu peux maintenant importer des prospects depuis n'importe quelle page.
Comment utiliser :
  1. Va sur une page avec une liste de prospects (CRM, portail, tableau...)
  2. Clique sur l'icône Prospector dans ta barre d'extensions
  3. Clique Scanner — les contacts sont détectés automatiquement
  4. Clique Importer dans HubSpot
