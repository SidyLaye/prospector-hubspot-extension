Connecte ton compte HubSpot
Crée une application privée HubSpot pour obtenir ton token d'accès. C'est gratuit et ne prend que 2 minutes.
Commence par pat-eu1- (Europe) ou pat-na1- (USA)
Comment créer ton token :
  1. Va dans Paramètres HubSpot → Intégrations → Applications privées
  2. Clique Créer une application privée
  3. Donne un nom : Prospector Extension
  4. Dans Portées (Scopes), coche :
crm.objects.contacts.read crm.objects.contacts.write crm.schemas.contacts.read

  1. Clique Créer l'application et copie le token généré ici

Prospector est connecté à ton HubSpot. Navigue sur n'importe quelle page avec une liste de contacts et clique sur l'icône de l'extension pour lancer l'import.

Comment utiliser :

  1. Va sur une page avec une liste de prospects (CRM, portail, tableau...)
  2. Clique sur l'icône Prospector dans ta barre d'extensions
  3. Clique "Scanner" — les contacts sont détectés automatiquement
  4. Clique "Importer dans HubSpot"
